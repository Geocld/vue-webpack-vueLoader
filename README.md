# vue-webpack-vueLoader
搭载了webpack、vue-loader的vue组件开发脚手架，详细tutorial链接[here](http://geocld.github.io/2016/03/14/vuejs_webpack/)。

```
# install dependencies
npm install

# serve with hot reload
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
